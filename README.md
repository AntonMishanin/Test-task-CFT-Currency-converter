# Currency converter
Currency converter is a small test task application, based on Clean Architecture and MVP. 

## Tech stack
- Clean architecture.
- MVP+Moxy for presentation layer.
- Fetching data from the network with Retrofit+Rx.
- Realm+Rx.
- Dagger.
